# bt_api_coincheck

Coincheck exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_coincheck
```

## Usage

```python
from bt_api_coincheck import CoincheckExchangeDataSpot, CoincheckRequestTickerData
```

## Features

- REST API support
- HMAC-SHA256 authentication
- Symbol format: `btc_jpy` -> `BTCJPY`
